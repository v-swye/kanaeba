开始：基本操作

启动git服务

首先git init在当前目录启动git，读取隐藏的.git文件夹。
一个本地仓库可以接多个远程仓库，代码为：
git remote add [remotename] git@github.com:[username/repository.git]
查看当前目录下的远程连接：git remote
删除远程连接：git remote remove [remotename]

添加跟踪文件，commit与push

在commit前，需要先添加要track的文件到暂存区：git add .
查看状态：git status，绿色表示和暂存区track的文件一致，红色则表示存在未track的文件。
然后commit: git commit -m “comments”
修改commit内容：git commit --amend
在多人合作时，应commit后先pull更新本地到最新版本。
最后push到对应branch，一般为master：git push -u [remotename] master。如果要覆盖已有的远程仓库，将参数调整为-f即可（尽量不要使用）。

回退commit

查看commit记录：git log
回退：git reset --soft HEAD~1，1表示回退到上一步，2表示上上一步，依次类推。注意不要使用--hard，否则会修改覆盖本地仓库的文件。

拉取远程仓库至本地：fetch, merge与pull

pull=fetch+merge，所以安全起见应该先使用fetch，检查差别后再merge。

关于git fetch更专业的表达为git fetch remotename 远程分支:本地分支

退出git

输入:q。

文件查看与删除库

查看本地仓库中的文件：ls -a
删除本地仓库（不删除文件，相当于只是取消git）：rm -rf .git
删除本地文件：rm -rf filepath

分支操作

查看本地所有分支：git branch
新建本地分支：git branch newbranch
将本地分支提交到远程并在远程新建对应分支：

查看本地+远程分支：git branch -a
切换远程分支：git checkout -b [remotename] [remotename/branchname]
切换本地分支：git checkout branchname
删除本地分支：git branch -d [BranchName]
删除远程分支：git push origin --delete branchname
合并分支：首先切换到要保留的主分支branch1，然后git merge branch2。

Posted on 2020-05-03 | In 编程与技术

计量理论知识

最近帮导师做了份编程作业的答案，也是第一次学习实际implement做ARIMA和简约VAR模型。机会难得，正好系统复习和学习一遍时间序列的基本建模方法。

平稳时间序列

定义

在计量经济学中，时间序列建模的首要条件是平稳性（stationarity）。平稳时间序列的基本定义是，时间序列的统计性质（statistical properties）不因观测时间的不同而发生改变。弱平稳时间序列（weak stationary）数学语言表述为：

  1. First-order moment is constant, i.e. $E(y_t)=\mu$
  2. Second-order moment is constant and finite, i.e. $Var(y_t)=\sigma^2$
  3. Auto-covariance depends only on the interim period length, i.e. $cov(y_t,y_{t-j}) = cov_j$

常见的非平稳序列特征包括趋势（trend）和季节性（seasonality）。不过，并非所有看似有周期性的序列都是非平稳的——如果周期的长度是随机的。

检验时间序列的平稳性

一般常用Dickey-Fuller test检验平稳性（即ADF检验）。原假设为非平稳时间序列，存在单位根，故检验的目标为p值应小于设定的显著度。

非平稳序列的转换

常见的转换方式包括去趋势、季节性差分、N阶差分和对数化。

  1. 去趋势：观察趋势的类型，最简单的趋势为线性趋势。如为线性趋势，则先regress y on 1,2,3,…，然后取$y’_t = y_t - \hat{y}_t$，其中$\hat{y}_t$为线性趋势模型中的fitted value。同理，对于非线性趋势，可以使用多项式模型进行fit，从而remove掉趋势。
  2. 季节性差分：如季节性为m期（例如，一般季度周期下m=4），则$y’_t = y_t - y_{t-m}$。
  3. N阶差分：对时间序列进行N次差分，如N=2，则$y’_t = y_t - y_{t-1}, y’’_t =y’_t-y’_{t-1}$
  4. 对数化：顾名思义，取自然对数，$y’_t = ln(y’_{t-1})$。

Modeling ARIMA/ARMAX with Python

ARMA模型需要设定两个参数：p和q。如何调整这两个参数呢？常见的方法包括：

  1. ACF和PACF（Partial Auto Correlation Function）
  2. 结合其他基准：信息准则（AIC，越小越好），显著程度等

ACF & PACF

ACF（autocorrelation）计算公式为：$ACF_h = \frac{Cov(y_t,y_{t-h})}{Var(y_t)}$ （因为弱平稳条件下$SD(y_t)$ 不随t变化）。
PACF（partial autocorrelation）描述将其他变量的相关程度剔除后的剩余部分的自相关性质。这么说起来有些抽象，下面举例说明：$y_t$和$y_{t-1},y_{t-2}$相关，而$y_t$和$y_{t-2}$的相关性之间就存在$y_{t-1}$的干扰。把$y_t$和$y_{t-2}$中与$y_{t-1}$不相关的部分剔除出来，再计算得到的相关系数就称为偏相关系数。计算PACF的方法就更加明了了——做线性回归，取residuals计算相关系数。这样的相关系数在数学语言上用条件相关系数来表示：

读书笔记：玫瑰战争简史

Posted on 2020-03-08 | Edited on 2020-03-14 | In 读万卷书

对我这样的英国历史白痴而言，这本书实在算不上科普，更多像是为历史爱好者准备的学术论文式作品；但也正是因为这种严谨的学术性和深刻的政治、经济、文化及社会变迁的全方位分析，促使我尽可能理顺这段历史过程，走过15世纪的英格兰王朝。玫瑰战争始于理查二世被亨利四世推翻，到亨利五世、亨利六世、爱德华四世、爱德华五世、理查三世期间，代表约克家族的白玫瑰与代表兰开斯特家族的红玫瑰屡次交锋，最后于亨利七世与约克派的伊丽莎白的婚礼中画上句号，开启都铎王朝。

历史时间轴

对于（业余）历史研究，我想首先不得不做的是梳理时间轴，理清各个事件的先后顺序，从而在探讨各个事件发生的可能“条件”和事件之间的社会变迁。玫瑰战争时期，英国的王室更替大致如下展开：

  • 1399年：理查二世（约克家族）被亨利四世（兰开斯特家族）夺权；
  • 1413年：亨利四世的儿子亨利五世继承王位
  • 1422-1461年：亨利六世九个月即位，初期英国的摄政贵族为格洛斯特公爵汉弗莱。亨利六世缺少王室教育、无心政治，但创办了伊顿公学和剑桥大学国王学院
  • 1461年：爱德华四世夺权
  • 1470年：在沃里克伯爵和克拉伦斯公爵的帮助下亨利六世复辟
  • 1471年：爱德华四世再次夺权，亨利六世死于伦敦塔
  • 1483年：格洛斯特公爵摄政辅佐时年12岁的爱德华五世（同年即失踪），格洛斯特公爵继位为理查三世
  • 1485年：亨利都铎从理查手中夺得王位，终结玫瑰战争，开启都铎王朝

夺权、交战与社会变迁

在短短不到一百年见，英国见证了多次王权更换，约克派和兰开斯特派之间到底是为了什么如此针锋相对？实际上，和传统封建帝国不同，英国是一种“混合君主制”，即国王的决策仍需要受到议会（一定程度上）的约束，而英格兰绅士们的骑士精神一向以捍卫王室尊严、以为王室而战为荣。因此，认为内战是源自对王位的“觊觎”的想法并无合理之处；然而，篡位终究有其动机，而每一次王权更替的动机中，都展现了当时英国统治阶级和民众结构间暗藏的变迁。

15世纪的兰开斯特王朝背后，一直依赖于一种新的社会背景：王朝建立在民众支持的基础之上。例如，玫瑰战争的起因要追溯到1399年，理查二世被兰开斯特公爵柏林布鲁克的亨利推翻。英格兰的法律是议会基于人民的意愿制定的，而理查二世认为自己凌驾于法律之上，王权高于法律。在议会的推动下，亨利四世主导了王权。这种合法性存在漏洞的王权是十分脆弱的。当亨利四世为了培养支持者，向贵族慷慨赠予金钱和土地，同时要求议会提高征税时，他便触犯了帮助他上台的盟友。与四世相比，亨利五世被称作是“英格兰最伟大的君主”，许多现代历史学家都赞同麦克法兰的观点：他的统治证明只有强而有力的君主才能有效调动政府的资源，并且确保国内的公平正义和强大的财政实力，同时施展成功的对外政策。

亨利五世在国防建设、财政管理和司法方面做出了卓越贡献，但作者认为，真正重要的是亨利五世在对法作战中展现的杰出才能。在1420年5月签订《特鲁瓦条约》（treaty of Troyes）后，亨利五世成为法国国王查理六世的继承人，并与查理六世的女儿瓦卢瓦的凯瑟琳成婚，王位继承权得到了进一步的巩固，成为中世纪君主的典范。

可惜的是，亨利五世并没有来得及巩固对法统治，九个月即位的亨利六世自小缺乏王室教育，不具有与君主之位相当的能力和抱负。在格洛斯特公爵汉弗莱摄政期间，整个国家的重心都在寻找着即使缺少君主，社会也能继续运转的制度和方法。这种稳定团结的局面并没有得到长期维持。
英国王室具有一种资本主义萌芽的特性，即王室自负其财政，而为战争征税需要得到议会的批准；国王与贵族之间的权衡除了私人或血缘的亲密关系外，更多仰赖一种纯粹的利益关系，即国王提携贵族，而贵族及其扈从继续为国王提供服务。因此，麦克法兰认为，当懦弱的君王不足以维持这张错综复杂的权力蜘蛛网，王室陷入财政危机债台高筑时，贵族的忠诚与否依赖于与国王之间的亲疏、逐渐走向拉帮结派，政治危机随之而来。而在亨利六世执政期间，王室再次陷入了财务危机，国王在英法作战中又惨遭战败。战败的消息直接激发了民众对王室的不满和反对，而这动摇了整个兰开斯特王朝的根本。

缺少政治话语权及背景的国民无法与国王就他的无为进行当面对质，他们只好寻找替罪羊来解释国家不断加深的政治、军事和财政危机。

亨利六世的得力助手——萨福克公爵便是“推举”出来的替罪羊。他被以叛国罪指控，但亨利六世否定了这一点。不过，他还是没能逃过临时审判，最终死于民众的乱刀之下。这一事件也体现了当时民众参与政治的激昂情绪。1453年夏季，亨利六世精神失常，为风雨欲来的政权危机点燃了最后的导火索。

仅从国王和贵族之间的角逐来看，政权的稳定在于强大的君王和强势大臣各自出现的时点。懦弱无能、优柔寡断的君主，再加上横行无忌、野心勃勃的贵族，足以导致秩序的混乱。正如麦克法兰所说：

“实际上，只有无能的君主才会惧怕‘强势大臣’。君主之所以无能，因为他本身就不适合当君主，而不是因为王位合法性问题和他在统治谋略方面的欠缺。”

约克公爵理查以“改革的拥护者”的身份被民众推到政治斗争的前线。但是，百姓的支持并不足以破坏亨利六世身边或是忠心或是远离政治的贵族网。1451年5月，下议院支持正式立约克公爵为王位继承人，但是却立刻遭到国王和贵族的反对——亨利六世解散了议会。就在约克公爵以“为民改革”（再一次体现了兰开斯特王朝的政权基础）争取王位失败时，亨利六世精神错乱了，这使得约克公爵以护国公和英格兰保卫者的身份开始管理国家。在此期间，亨利六世的王后玛格丽特为了自己儿子威尔士亲王的继承权，一直活跃于组织反叛；1454年，亨利六世再次恢复意识，直接熄灭了约克公爵的王权希望。

玫瑰战争终于拉开了序幕：约克公爵开启暴力反抗君主的先河，但在1459年路孚德桥的对峙中，由于约克派士兵的人数劣势及临阵倒戈，几位领导的贵族旋即退回拉德洛，留下群龙无首的军队，而约克公爵在次年12月意外战亡。
亨利六世对反叛的勋爵们采用的安抚政策一定程度上舒缓了当时的危机，但是如果这种暂时的平静得不到有效的王权领导力支撑，那么很快动乱就将找上门来。正如他最初在“爱心日”对约克公爵的安抚一般，只凸显重臣之间的个人恩怨而非更深层次的权力问题将埋下未来更大爆发的种子。1461年，伦敦市堵住兰开斯特家族回归的脚步（主要原因可能是惧怕这些一路烧杀抢掠的北方人），转而迎接已逝约克公爵的儿子爱德华进驻伦敦，成为爱德华四世。

爱德华四世实际上得到的承认只是在血统上具有“王位继承权”，而亨利六世因为违反协议理应得到废黜。不过，官方的政治宣传却大打“民众推举”之牌，再一次充分体现了兰开斯特王朝来自“人民”的王权基础。
爱德华四世在玫瑰战争中最大的一场战役——陶顿战役——中，取得了胜利。尽管兰开斯特军队在实力和人数上都胜过约克派，但新国王的军事领导能力、亲自参与战斗鼓舞士气、约克派“强势大臣”勋爵们对公共资源的控制和充分利用、羊毛商人提供的私人借贷和民间支持都为约克派的胜利奠定了基础。然而，爱德华四世的一大失误在于未能与强势大臣——沃里克伯爵——和平相处。再一次，“只有懦弱的统治者才会害怕强势大臣”，爱德华四世对这个昔日功臣的防范和屈才使后者积怨渐深。而国王自作主张的滑稽婚事更是使贵族们大跌眼镜：在婚后一年多，爱德华才宣布自己已经与约翰·格雷爵士的遗孀、兰开斯特派里弗斯勋爵理查·伍德维尔的女儿伊丽莎白·伍德维尔成婚，不顾沃里克伯爵的强烈反对。伊丽莎白家族人口众多，“外戚干政”影响国内的政治平衡，而且也断绝了其他各国原本想要和英国携手联姻的意愿。最终，沃里克联合玛格丽特王后复辟亨利六世。不过，爱德华四世很快反击，将亨利六世再度送回伦敦塔，与伊丽莎白王后重聚。这一次，爱德华选择了吸取教训——立即处死亨利六世。

1469～1471年间所发生的历史事件同时体现了中世纪后期英格兰政体的优点和劣势。一方面，王室不仅在财力方面相对较弱，而且要依靠权贵来分担军事防御和司法领域的权力。这种局面为“强势大臣”的涌现提供了机遇，也给国王的权威带来了威胁。但他最大的失误之处仅仅在于忽略了沃里克伯爵的政治期许，打击了他的雄心壮志。

很少有人能够想象到在爱德华四世去世10周多一点的时间后他的儿子即被废黜，并被宣布其王位继承权不合法（爱德华四世的婚姻不合法，所以他的子嗣便都属于私生子），而摄政辅佐爱德华五世的格洛斯特公爵才是约克公爵理查真正的继承人，他将登上王位并成为理查三世。理查三世的加冕仪式并不是没有不满的声音，只不过场地附近安排的4000名北方士兵足以镇压一切反对。

赞同理查三世对爱德华四世的两个儿子的谋杀的人少之又少，同时，篡位行为本身进一步削弱了国王的权威，使得人们对王权的挑战变得可能，甚至是可能性十分之高。最终，理查三世被受法国国王和私人借贷支持的亨利·都铎击败，后者通过迎娶了爱德华四世的女儿约克的伊丽莎白结束了两大家族最后的斗争，为玫瑰战争拉下落幕。

1485年8月，亨利·都铎成为英格兰国王，其夺取王位的过程凭借的是好运势（正如理查三世及其父亲约克公爵以身家性命为代价所发现的那样：战争充满了风险）再加上政治国家中很大一部分人的远离政治。颇为自相矛盾之处在于，这种大部分人对政治进程参与的缺乏，将使亨利·都铎得以建立起一座崭新的王权大厦。这是一种带有明显都铎王朝风格的管理方式和政治文化，在接下来的一个多世纪里，这种政治文化会历经比整个15世纪更为激进的历史进程而得以延续。

通过梳理这段历史，我想，玫瑰战争时期的政治和社会特征可以总结如下：

  • 整个兰开斯特王朝的王权基础来自人民的推举和支持
  • 国王的领导能力、军事实力和财务管理能力与其能否在动乱中维持一种和平的局面息息相关
  • 贵族呈现远离政治的倾向
  • 民众的政治参与热情有所提高
  • 年幼的继承人与强势摄政大臣的组合有极高的篡位风险；但篡位者需要极大努力才能获得民众对王权的承认
  • 安茹的玛格丽特王后尽管最终复辟失败，但是不失为一名领导力优秀、坚定、热情且有力的女性

读书笔记：寂静的春天

Posted on 2020-02-22 | Edited on 2020-02-25 | In 读万卷书

春天本是一场盛宴。万物复苏，草长莺飞，一切从冬日的枯寂、沉睡与冰封中醒来。连肺炎的鸡血广告词都是“没有一个春天不会到来”。然而，若春与冬的分界点日渐模糊，最终世界归于永久的寂静——那样的日子，并不是危言耸听；它正在逐渐逼近，且经人类之手应邀而来。

不是魔法，也不是敌人的活动使这个受损害的世界的生命无法复生，而是人们自己使自己受害。

控制自然

科技的发展，特别是化学的飞速发展，将人类的野心展露无遗。杀虫剂的发明不仅让饱受虫害侵扰的农民看到了“希望的曙光”，更让任何一名“关心”生态的人品尝到亲自改造自然的快感。然而，“试图解决某个问题但随之却带来一系列灾难，这问题是我们现代生活方式的伴随物”。人们没有想到，人类激进的步伐早已超越了以千年为最小时间单位的自然均衡过程。一旦这种平衡被破坏，那么在自然界将其彻底修复之前，保守估计下我们这一物种也需要经历数百代的更替。可悲的是，我们曾在这一失衡的路径上加油助威、呐喊着进军；而现在，刹车的脚还犹犹豫豫。或许，重温过去的牺牲，可以警示我们下定决心，并意识到——“控制自然”这个词是一个妄自尊大的想象产物。

恶果初现

虫害一直是令全球农业头痛的课题。大量的化学药物控制仅仅取得了有限的胜利，但却对生态环境造成了更大的威胁。
杀虫剂并不会仅仅出现在挣扎在死亡痛苦的虫体上，它还会停留在树叶、灌木上。更不用提那些通过飞机喷洒的化学物质——它们挂在屋檐下、擦过私家车的侧身、侵蚀经过的鸟类、落在人类裸露的皮肤上。除草剂也并没有比杀虫剂安全多少。危险的是，由于作用机制适用于多种生物系统，这些化学物质作用的目标并不局限于最初想要消灭的昆虫或是杂草。
鸟类是能观察到的最先受影响的物种之一。“谁杀死了知更鸟”不仅是广受好评的一本反种族歧视的小说，更是实际存在过的生态悲剧。曾经，对于千百万美国人来说，第一只知更鸟的出现意味着冬天的河流已经解冻。知更鸟的到来作为一项消息在报纸上报道，在坊间传唱春天的消息。然而，现在的美国再难寻觅知更鸟的踪迹==？？？==。而作为昆虫的天敌，数量骤减的鸟类意味着生命力旺盛的昆虫更加泛滥，这与最初喷药的初衷背道而驰。除了生物本身，药物还在胚胎和蛋黄中贮存，导致有些地方的鸟儿无法完成孵化、或是孵化后不出几天即迎来死亡。渗入土地后冲入河流的化学物质进一步威胁着鱼类的生命，特别是那些有在上下游之间来回巡游以完成产卵习性的鱼类。除了直接致死外，滴滴涕可以使鱼眼变瞎的事实已在许多研究结果中报道。在温哥华岛北部对喷药进行观察的生物学家报告过，原来很凶猛的鳟鱼眼睛上已蒙上了一层不透明的白膜，这使它们的视力减弱或完全丧失。
尽管最初设计喷洒时，药物剂量按照规定严格执行。然而，这类人工合成的化学物质可以在自然界存在多年，而年复一年执行喷药计划使每平方米土地上的累积浓度不可估量。同时，这些化学物质进入土壤内诸如蚯蚓等生物体内、或是随着水流在鱼类体内沉积。随之，它们开始在食物链中的传递和放大旅程。当进入位于食物链较顶层的生物体内时，其累积浓度可能早已超过致死量。
而实际上，或许是得益于快速的繁殖速度，昆虫很快就能对这些杀虫剂产生耐药性。与杀虫剂产生的微弱效应相比，其成本却极其高昂。除了研发和制药的高额成本之外，消费者最后还要增加纳税以支持制订政策机构来查证落实他们是否会得到致死的剂量。其结果是，倒霉的消费者付出了税钱，且仍然在摄入不受人们注意的那些毒物。

自食其果

最终，人类创造的化学合成物将反噬到人类自己身上。通过漫长的进化演变，人类凭借自身拥有的独特的遗传物质成为今天的人类，在自然界中享有独一无二的智慧。然而在当前，人为因素所引起的危害已成为我们时代的一种威胁。一种理论认为，化学物质可以通过抑制细胞正常的呼吸作用致癌。瓦尔堡相信，无论放射性致癌物还是化学致癌物，都是通过破坏正常细胞的呼吸作用而剥夺了细胞的能量。这一作用可以由经常并重复地给与小剂量暴露而造成，且一旦造成，就不可恢复。细胞不再能进行产生大量ATP的有氧呼吸，而是转向效率低下、类似发酵作用的呼吸方式。借助于发酵作用而维持生存的斗争经常会继续一段很长时间。这种发酵呼吸方式通过以后细胞分裂而传递下去，以至于后来产生的细胞全都具有这种非正常的呼吸方式。这也解释为了为什么小剂量的频繁暴露有时比一次性大剂量暴露更加危险：一次大剂量中毒可以立即杀死细胞，然而小剂量却容许一些细胞存活下来，然而这些细胞却成为未来癌变的导火索。
另一些研究者认为癌细胞中数目紊乱、易受损害的染色体提供了另一种可能的影响渠道。阿尔伯特·黎凡和约翰·J·倍塞尔通过对染色体的变态发展研究得出结论：染色体的异常变化发生在恶性病变之前。而化学物质导致的染色体异常变化也可以通过一代代分裂传承累积，最终开始不规律的增生。
通向癌症的道路也可能是间接的。洛克菲勒医学研究所的一个研究人员发现，由于疾病而使肝脏受损的兔子表现出子宫肿瘤的高发病率，研究人员认为子宫肿瘤高发的形成是因为肝脏已不能再抑制血液中的雌性激素，以致于“最后这些肿瘤演化到癌变的水平”。而氯化烃就是能损害肝脏的物质之一。即使是很低的氯化烃摄入量就能引起肝细胞变化，同时引起维生素B的损失，而这种维生素对抵制癌症有重要保护作用。
正如杀虫剂本身的机能原理是对昆虫的神经系统产生作用一样，除了慢性致癌隐患以外，已有研究将杀虫剂和精神疾病联系起来。一些精神病例有着长期暴露于有机磷杀虫剂的病史。而一些研究者用自己做实验，最终导致瘫痪或是长期精神不集中的情况并不少见。

警示

由此可见，杀虫剂的使用可谓是以极其高昂的成本换取微乎其微的效益，而这种效益甚至在耐药性的“帮助”下不可持续。对于虫害、杂草的治理，防治历史警醒我们——不要追求快速成效，赔了夫人又折兵。否则，迎接我们的是一定会到来的春天，可惜却再也见不到繁花与翠柳，而只有一片寂静。

寻找更好的解决途径

那么，如果没有化学试剂，我们便只能选择任由虫害和杂草侵入，影响粮食和畜牧生产吗？答案并不是非黑即白的。正如英国生态学家查理·艾尔登所说，关键是在于“保持多样性”。生物防治是古老、有效而又不破坏生态平衡的最佳方法之一。例如，向虫害聚集区引入其天敌，将昆虫的最终数目维持在一个较低但可控的水平范围内。另外，不育法也有了部分应用。1916年，人类便发现了X射线照射下可能导致昆虫不育。如果有可能使很大数量的昆虫不育，并把它们释放出去，使这些不育的雄性昆虫在特定情况下去与正常的野生雄性昆虫竞争取胜，这个种群就濒临灭绝了。当然，并非所有昆虫都是这种技术的合适对象，这种技术在很大程度上要依靠对昆虫生活史的详情细节、种群密度和对放射性的反应的认识。细菌杀虫剂也已经被尝试着应用于各种各样的蔬菜上。与化学药物相比，昆虫病菌具有高度专一的有点。除了对其要作用的对象外，这些病菌对其他所有生物都是无害的。
而作为与科研距离较为遥远的普通人，需要建立的是一种对生态环境和人类之间相互作用的正确认识。例如对于癌症，就像减少传染病的病灶比治疗病人更能取得成效一样，防患于未然、减少杀虫剂这类无形的致癌因素势在必行。而在更高层面，这就要求我们充分了解各类化学制品的危害，而不是在肆意使用杀虫剂时无视上面的各种提示小字。在哲学观点上，则必须放弃人类优越的观点，放弃“控制自然”的狂妄想象，转而去尊重自然、尊重生命。科学上需要的是谦虚谨慎，没有任何理由可以骄傲自满。

生命是一个超越了我们理解能力的奇迹，甚至在我们不得不与它进行斗争的时候，我们仍需尊重它。

读书笔记：猿猴的把戏

Posted on 2020-01-27 | In 读万卷书

与《黑猩猩的政治》不同，本书并不是一本灵长类观察记录，而更多着墨于猿猴行为与人类社会之间的映射关系。作者首先以对有趣的“电梯困境”观察展开。

当人们一起搭乘电梯时，他们对待别人的方式依然表明人们对于自己的人身安全有着严肃的关切。如果电梯里很拥挤，那么每个人都会静静地站着，眼睛望着天花板、地板、他们的手表或者电梯的按钮，就好像他们从前没见过这些东西一样。如果是两个陌生人一起搭乘电梯，那么他们会站得尽可能的远，避免面对面地跟对方有眼神接触，避免做出什么突然的动作或一不留神发出声响。

作者认为，电梯中的大多数行为并非出于理性，而是一种伴随着经历过类似场景的基因进化而来的本能反应。因为我们的大脑认为存在危险，所以会即时启动自我保护的行为。对于大多数人来讲，如果实在无法避免眼神接触，那么在电梯中通常会有一方先展现友好的姿态，例如微笑和闲聊，从而化解这种本能的危机感。这一点或许也有一些性别上的区别，例如学者研究发现，男性在面对社会压力时会倾向于表现出一种“战还是逃”的反应，而女性则往往表现友善行为，试图化敌为友。不过，整体来讲，这两种策略在现实中是相互交叉的。
灵长类动物在类似场景表现出相似的行为。当两只猕猴被关在一个小笼子里时，它们会想尽办法避免发生打斗，例如静坐或漠不关心；时间一长，紧张的气氛逐渐凝聚起来，它们就需要借助于沟通，同时一只猴子有必要让另一只猴子明白自己没有意图伤害它。
由此我们可以作出一定的猜想：即人类和灵长类动物之间的行为或许存在某种相似的模式。更为特殊的是，人类和灵长类在生存和繁衍方面的成功更大程度上依赖于同类中其他个体的行为，无论是通过竞争还是合作，而这一点在其他动物群体中很少出现。研究灵长类的行为或许有助于揭示人类行为的谜团。

裙带主义

指向家庭的自私基因

一个人能在多大程度上实现他的社会目标，除了个人能力之外，一般也取决于他的团队整体能力以及他的团队能给他多大的支持。裙带关系便是这种支持的主力军（另一种是政治）。从生物学的角度来看，裙带主义是指向家庭成员的利他主义。猴子和人类都会乐意帮助亲戚，因为亲戚跟他（它）们拥有某些共同的基因，通过利他行为，他（它）们就增加了这些基因在群体中保留下来的机会。因此，从另一个角度来看，这种行为实际上并不是真正的利他，而是为了延续自身基因的“自私”行为。

裙带社会的健康运行

裙带社会对于支持理想化完全平等的人士来说或许是悲观的。或许，《动物农场》中的那句“所有的动物都平等，但有些动物比其他动物更平等”暗示了生物社会的残酷现实。而且，裙带主义的受益者从来都不是消极无为的旁观者，当他们意识到裙带关系可以为他们带来利益时，他们会毫不犹豫地利用机会。和支配关系一样，裙带社会早已根植在人性深处，我们要做的鞭尸维护它的健康运行。对此，作者提出了三点建议：
一、 不要让资助者难堪。因为只是为了增强基因的延续而放弃雇佣能力好得多的候选人，显然无益于社会的演进。
二、 不要让自己难堪。
三、 传承裙带主义：如果你是自己父母裙带主义援助的接受者，用你对自己孩子的裙带主义行为表达对父母的感激。

支配关系

支配结构

在作者眼里，人类社会的核心是由一个个支配结构建立起来的。人类的行为往往会随着他们在这段关系中所处的地位而大不相同。例如，在电子邮件往来过程中，处于从属地位的人显然不得不多次斟酌用词、用大段大段的文字争取换来支配者寥寥数语的回复。支配关系存在在人类可见的一切关系中，例如亲子关系、手足关系和婚姻爱情关系等等。许多稳定的手足关系都出现在年龄差距较大的孩子中，这种年龄差为长子提供了绝对优势，顺理成章地占据并维持着支配者的地位。而在爱情关系中，最稳定的浪漫关系和婚姻关系从一开始就有着明确的支配结构。伴侣中的支配者决定所有的事情，而从属者默许同意，扮演着支持性的角色。

自然选择有一种“简约式”的倾向，如果能以简单的方式解决问题，它绝不会走向更复杂的一种。因此，支配关系在个体之间建立起明确的阶梯，确保在支配结构稳定的一段时间内，最快地解决双方之间的潜在冲突：不是打上一架，也不是经历漫长的谈判，而是十分简单明确的结果——支配者得到自己想要的，而从属者则没有。但这种粗暴的解决方式并不是没有代价的，而且，如果遇上一个冷酷的支配者，那么这种代价将全部由从属者承担；有的支配者会相对仁慈，让这些失败者也可以得到些许好处，或是加以安慰，使二者之间的利益冲突相对弱化。

正是因为成为支配者的诱惑是如此之大，支配关系并不总是稳定的。正如书中所说的那样，“人们不会为这件事或那件事而斗争：人们是在为这件事或那件事上谁说了算而斗争。换句话说，我们在为决定支配权而斗争。”为什么支配者能成为支配者，而从属者又甘愿付出代价？一段稳定的支配关系需要具备什么要素？什么时候从属者会试图挑战支配者，逆转地位？对于支配关系及其动态变化的分析，经济学早已为我们提供了一套较为完善的方法，即博弈论。一旦使用博弈论来理解支配性，我们就会意识到，支配关系真实存在，而且有且只有一种。

关系建立的策略

在基于博弈论框架进行分析之前，首先需要指明每个个体所具有的三个要素，即成本、收益和禀赋（在本文的特定背景下，作者称其为resource holding potential，简称RHP）。从属者之所以愿意成为从属者，主要是因为成为支配者需要付出巨大的代价，而作为从属者能够减少损失；而支配者自然理所当然地享受自己的特权，而不必付出什么。勒庞在《乌合之众》中无不讽刺地批判道，始终支配着群体心灵的是对被奴役的渴求。显然，如果一定要分析成本和收益的话，那么服从意味着群众不必大费周章地去自行推理，而只需简单接受他人的断言。

在两个此前从未谋面的博弈者第一次见面时，他们会立刻开始评判两者RHP（例如体型）之间的差距。如果这种差距十分明显，那么支配关系的建立无需试探。不过，大多数配对之间RHP的差距并不是那么显而易见，此时，支配结构的建立可能会需要经过一个漫长的过程，二者之间选择战斗或是谈判，来改善双方之间对于RHP信息了解程度的不对称性。

如果采用战斗的策略，对于猿猴来说结果的主要影响因素包括体型、身体力量和冒险意愿。用博弈论的术语来讲，物品价值对两个竞争者的非对称性被叫作支付非对称性（asymmetry in payoffs）。拥有更强冒险意愿或战斗精神的个体，通常是能在战斗胜利中得到更多好处的那个竞争者。猴子倾向于选择类似这样的策略：该策略的收益高于成本，且该策略的收益成本比率（benefit to cost ratio）高于其他策略。

沟通/谈判是向对方直接供认非对称信息的方式。鉴于通过行为沟通RHP和动机有利可图，自然选择青睐具有这种功能的特定行为信号的进化。这些信号被称为行为展示，以便跟RHP的身体展示相区分。不过，谁也不能保证沟通是真诚的。鉴于成为支配者的利益诱惑是如此之大，个体也许会选择欺骗，发出误导性的信息。当然，自然选择也会鼓励怀疑能力以及探测虚张声势的能力，因为相比轻信的个体，具有怀疑精神的个体更可能成为支配者。

最后，维持支配关系的行为也与二者的RHP差距有关。力量差距明显时，从属者长期处于恐惧中，因而往往会选择主动示好、向支配者表达屈服的策略；而在RHP差距开始缩小时，支配者则尝尝通过恐吓等行为来迫使从属者确认自己的地位。不过，即使在非常不对等的支配关系中，从属者始终有可能在某一天发动反叛。这是因为，两个个体之间的非对称性可能随着情境和时间的变化而改变。

正确认识支配关系

作者认为，支配性处于人性深处，要幻想一个完全平等、没有支配结构的社会是不现实的。不过，我们也没有必要以悲观的态度来看待这样的“不平等”——毕竟，自然已经这样演化了上万年。个体在适当的机会下去竞争支配性，冥冥中促成了进化的演变。

我们不能阻止一些人成为支配者，但我们可以教育他们，让他们知道支配意味着责任。支配者拥有领导义务，因为被支配者为他们的成功添砖加瓦，支配者有理由对他们更好一点，更宽容、更慷慨、更仁慈。毕竟，这个世界上不存在永恒的支配——当时机到来的时候，我们必须做好准备。

竞争

无论是人类还是灵长类，对于一个群体来讲，成员的进出往往都是动态的。当一个个体想要进入一个新群体时，就会遇到许多困难，他/它会被视作外来者，经受群体的考验，最终决定留下及留下的地位，或是离开。对于外来者来说，面对竞争程度不同的群体，争夺社会地位的各种策略的成功率也是不同的。

竞争的分类

达里奥将群体的竞争性分为两类，一种是“美国式”的价高者得的比赛式竞争（contest competition），另一种则是“北欧福利式”的无激烈斗争的分摊式竞争（scramble competition）。卡雷尔·凡·斯海克等人建立了一个数学模型，用β来衡量群体内的竞争是更接近于比赛式（β=1）还是分摊式（β=0）。因此，β值体现了群体的支配结构是否明确存在，其具体数值主要与群体的规模、群体内雄性（资源竞争者）数量有关。在大规模的群体内，无论现有首领有多么强大，也不可能垄断所有资源，因此β相对趋近0；而雄性数量多的群体，要想成为首领，一般来说需要拥有更多的RHP，也就具有更大能力垄断资源，β趋近1。
新个体进入群体的策略也会受β的影响。当β趋近于1时，抢占高地位的潜在收益十分诱人，因此新来的个体更可能采用高风险的策略。反之，试图挑战现有首领的收益并不高，新个体通常会采用默默迁徙的策略，然后根据论资排辈的原则等待地位的逐级提升。而且，由于分摊式竞争下论资排辈成为更有优势的策略，这种群体内首领也会相对年长一些。
β不仅影响群体的支配结构是否明确存在和新个体进入时的策略，还影响各级之间的陡峭程度（即高地位者对利益的垄断程度，或者说群体的专制程度）。在趋于比赛式竞争的群体中，高地位者对低地位者的态度更为“轻蔑”，而为了维持这种支配结构的稳定，或者为了改变支配结构，个体采用的联盟手段也会随之变化。

联盟

当周围没有足够的裙带资源可以利用时，要想出人头地，必须依靠政治。与人类类似，对猿猴来讲，想要维持跟某个目标之间的支配关系，或改变这种支配关系，不同的雄性可能会结成进攻性联盟。基本的进攻性联盟类型有三种：

  • 保守性联盟：结成联盟的两个雄性地位都要高于目标地位；
  • 过渡性联盟：联盟中一个雄性的地位高于目标，另一个雄性的地位低于目标；
  • 革命性联盟：联盟中两个雄性的地位都低于目标。

保守性联盟多出现在β值很高的专制社会中。这种联盟的意义主要是对低地位者不时进行威慑，起到防御的作用。另外，保守性联盟也是测试对方与自己合作的意愿，确保高地位者之间不会出现叛徒或怀有异心。过渡性联盟也主要在β值较高的专制社会中发生。考虑到这种联盟对高地位的一方并没有什么好处，自然地这种过渡性联盟往往是裙带主义的一面——帮助自己的家人提升地位。
最有意思的仍属革命性联盟。革命性联盟一般在β值居中的社会中出现，这是因为，当专制程度提高时，联盟更加有利可图，但风险也大大提高了，最终使其只能在β值未达到临界点的时候实现收益-成本比最大化。

合作

政治联盟的建立是一个动态的过程，而其核心要解决的问题便是“合作”。如何判断潜在合作对象，在合作之后又应该如何维持合作关系、保证对方坚守最初的承诺，是每个想要寻求合作伙伴的人需要关注的议题。

名声建立与欺骗

达成合作的首要要件是存在一致的目标。但是，我们无法事前完全掌握潜在伙伴的所有信息，而要想首先确认对双方的信任，人们最先依赖的信息往往是这个人的名声。社会心理学家认为，我们对于名声如此重视，是因为我们都在不断地寻求来自他人的认可和尊重，以便维持我们的自尊，或提升我们的社会地位。但是，自然界的现象早已暗示了名声背后的动机可能是更为自私的利益。经济学家提出，人们对名声进行投资是为了让他们的个人财政所得最大化，损失最小化，而进化生物学家则认为，动物这样做的目的是让个体适应所得最大化，损失最小化。

作者以“清洁工”鱼与其需要口腔护理的“顾客”之间的双生关系作为例子。研究发现，“清洁工”会尝试在小客户那里建立良好的名声，以吸引更大的客户；当他们成功竞标后，便凶相毕露，以客户的口腔作为营养来源。通过在低报酬活动中适当牺牲建立起名声，骗子清洁工就可以在其他更有利可图的情景中获得好处。人类也不例外，大多基金募捐者会告诉捐赠人其他人的捐赠情况，甚至公开捐赠人的慷慨等级，以加剧他们在名声方面的竞争力。但米林斯基指出，如果人类在一个博弈中建立的名声不能传达到另一个博弈中，那么人类很快就会对“无私奉献”丧失兴趣。因此，人们在未来的社会情境中是否能被识别出来、进而带来实际收益，直接影响着他们在情境中对名声的投资与建设。而传达名声的一种途径便是传言——或者说，注视——研究发现，感觉自己被人盯着或我们的身份被人知道的时候，人类就更可能进行合作。而这依然可以用经济学的基本原理来解释，即人们不仅考虑着现在合作对未来收益所带来的好处，还会考虑当下的欺骗行为会带来的可能惩罚和代价。总的来说，当人们选择合作时，会希望聚光灯都聚集在自己身上，传播自己的好名声；而在幕后，任何人都会更倾向于选择背叛。

当你正在从事要求双方合作的社会交易时，我的建议是这样的：让所有的灯都亮着，让其他人知道你正在盯着他们。

情绪

尽管人类和自然界的其他生物一样，通过潜在利益和违约代价来调整合作关系，但人类也通过独具的情绪和道德机制来确保承诺的执行。例如，违背承诺所带来的诸如内疚等负面情绪是背叛承诺的强大障碍；而因为坚守承诺而带来的正面情绪也有助于人们继续坚持。

爱情与承诺

与商业伙伴关系略有不同，恋爱关系中似乎有一种力量使人不计较可得的利益、损失或是情绪，只求双方维持一段稳固的关系、停止继续搜索更好的对象。一种可能的解释是，爱情是无法解释的，爱情的存在使承诺关系变得不再那么“冷漠”。但是，一个比较荒谬的问题可以基于此提出，即为什么商业伙伴不陷入爱情以解决他们的承诺问题呢？或许，浪漫关系并不是和商业伙伴关系一样的承诺问题。作者引入了进化方面的观点来提出一种新的可能：爱情的进化功能并不是用来解决合作关系中的承诺问题，而仅仅是用来鼓励男人和女人成双结对，以便一起照料孩子。人类幼儿的夭折率实际是很高的，而自然选择鼓励父母选择一起照料的孩子继续存活，达里奥甚至提出，人类在性器官大小、性激素水平等方面与其他灵长类的差异为减少雄性“涉猎”倾向、鼓励双方结成稳定配偶关系给出了证据。在大多数哺乳动物中，如果父母一方就可以负担起全部的抚养责任，就很难见到一对一的浪漫契约。而那些被认为是成双结对的哺乳动物都存在双亲照料；此外，这些动物寿命较长，幼崽的发育期比较漫长和缓慢。因此离婚更容易出现在这样两种情况下：一种是结婚几年之后依然没有孩子；另一种是最小的孩子已经有四五岁大，可以单亲抚养成年。

行为理论的过去、现在与未来

累赘原理

累赘原理由扎哈维提出，其主要观点是，便宜的信号容易伪装，而昂贵的信号需要的资源只有卓越的个体才拥有。因此，昂贵的信号能够传递这些个体诚实的信息。例如，雄狒狒会通过互相抚摸睾丸的方式确认对方是否对自己忠诚，与此同时，他们也将自己暴露在巨大的风险下；雄孔雀巨大的羽毛虽然可以在求偶中展现竞争力，但从纯粹的功能角度来讲，这种特征实际上对孔雀有害而无利，这种巨大的代价表现为一种累赘信号，暗示了雄孔雀足够优秀以承担这种代价，从而具有了内在的诚实性。

累赘原理不仅在生物行为方面有着广泛的解释性，也可以应用于许多社会现象。扎哈维认为，当个体表现出利他行为时，他并不是真的要去帮助他人。相反，他们通过负担累赘，对自己的能力和资源广而告之。人类的炫耀性消费或许也只是为了表现自己的强大。

累赘原理为社会行为研究提供了一种新的视角。在累赘原理的分析框架下，想知道一个同伴给出的承诺信息是否可靠，最有效手段就是给对方施加一个代价，然后评估对方“购买”这个代价的意愿。

人们可以承受来自好朋友的侮辱，但不愿承受来自陌生人的戏谑。因此，对另一个体施加不利影响，能够给你提供可靠的信息，让你明白你跟他的关系到底怎么样。

同伴选择

同伴选择理论认为，在双人囚徒困境博弈中，如果人们可以自主选择合作的伙伴，会变得更加信赖他人，加强亲社会行为表现。因为具有合作倾向的个体会相互选择，从而排除背叛者。在高度竞争性的环境下，倾向背叛的个体也不得不表现得更有合作性，以期加入合作、成为从中获益的一员。

错误匹配

错误匹配理论为人类的“非理性决策”提供了一定的解释。进化心理学家将进化而来的情绪反应与它们恰好被激活的陌生环境错误搭配的情况称为错误匹配。例如，尽管汽车撞死行人的事不时发生，但人们在穿过马路的时候依然常常缺乏警惕性。错误匹配的理论认为，汽车是一种新近的发明，因而人类还没有学会以适当的恐惧和焦虑来处理与汽车有关的危险情境。同样的情绪在不同的环境下，就可能启发出不同的行为。

情绪可以改变成本与收益的主观重要性，因此有时候会导致人们以“非理性”的方式行事。在一些情况下，情绪导致非理性决策主要是因为情绪和情境之间的不匹配。根据进化的观点，当下的情境设置越是远离原始的环境，由进化相关线索激活的情绪所引发的行为就越不可能是理性的。在另外一些情况下，人们做出了看似不理性的决策，并不是因为情绪干扰了他们的理性决策过程，而是因为情绪激活了认知下位程序，导致了与理性模型预测不同的行为决策。

不过，批判者表示，很多时候人类并不会利用手边所有可得的信息来试图进行理性决策。比起完整的决策过程，人们更倾向于一种快速的“启发式”算法，而这种快速的模糊过程或许也为解释非理性决策提供了一种可能。

行为同源性理论

行为同源性理论认为，行为特征或许与形态特征一样，在不同的有机体中表现出了同源性。社会行为从祖先遗传而来，经过分化和进化，即使形态上已然天差地别，仍然相对不变地保留了行为的基本类型。大量动物行为观察研究发现，在许多情况下，不同物种的动物在面临相似的环境问题时，会各自独立形成相似的解决方案，这种现象被叫作协同进化（convergent evolution）。
我们无法直接证明行为的同源性，但这种同源性为从动物中学习人类行为提供了方向。社会关系背后的许多规则和模式是进化过程的结果，而同样的进化过程已经在其他动物那里导致了相似的模式。本书即是从进化生物学和比较心理学的角度，论证检验人类社会行为的学术科普读物之一。

表格

基本表格结构

有了latex，做标准学术表格就容易得多了。首先，要指定一个表格模块：\begin{table}[ht]（这里ht是调整位置的选项）；然后通过表头栏设定各个纵列的格式，如：\begin{tabular}{m{5cm} lllll}或\begin{tabular}{c|c|c|c|c}，前者表示第一列固定宽度5cm，其余为靠左列，各列之间无分割线；后者表示每列自适应宽度居中，各列之间存在分割线，但无外部分割线。

随后即可开始添加表格本体的数据。一般来说，我们需要先画一根横线或是双横线，横线只需要写入\hline，注意横线无需再添加换行符\\，例如双横线只需要\hline \hline即可。
表的第一列往往是一句备注式的表格介绍，故我们需要生成合并单元格后居中的表头，如合并所有6个单元格：\multicolumn{6}{c}{这是一张测试表} \\ %换行
